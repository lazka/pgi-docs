Signal Details¶

Retro.CoreView.signals.controller_state_changed(core_view)¶
Signal Name:controller-state-changed
Flags:RUN_LAST
Parameters:core_view (Retro.CoreView) – The object which received the signal

The ::controller-state-changed signal is emitted when a key is pressed or released, mouse pointer is moved, or a mouse button is pressed or released.

Applications should not connect to it.
