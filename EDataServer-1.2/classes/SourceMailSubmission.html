set_sent_folder(sent_folder)¶
Parameters

sent_folder (str or None) – an identifier for the preferred sent folder, or None

Sets the preferred folder for sent messages by an identifier string. The format of the identifier string is defined by the client application.

The internal copy of sent_folder is automatically stripped of leading and trailing whitespace. If the resulting string is empty, None is set instead.

New in version 3.6.
