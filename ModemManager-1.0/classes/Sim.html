change_pin(old_pin, new_pin, cancellable, callback, *user_data)Â¶
Parameters

  • new_pin (str) â The new PIN code to be set.

  • cancellable (Gio.Cancellable or None) â A Gio.Cancellable or None.

  • callback (Gio.AsyncReadyCallback or None) â A Gio.AsyncReadyCallback to call when the request is satisfied or None.

  • user_data (object or None) â User data to pass to callback.

Asynchronously changes the PIN code in the SIM card.

When the operation is finished, callback will be invoked in the

thread-default main loop of the thread you are calling this method from. You can then call ModemManager.Sim.change_pin_finish() to get the result of the operation.

See ModemManager.Sim.change_pin_sync() for the synchronous, blocking version of this method.
