GoVirt 1.0 (0.3.7)¶

Parent Project:libgovirt (GObject-based library to access oVirt REST API)
Description:libgovirt is a library that allows applications to use oVirt REST API to list VMs managed by an oVirt instance, and to get the connection parameters needed to make a SPICE/VNC connection to them.
