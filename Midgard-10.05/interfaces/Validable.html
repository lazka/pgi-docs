is_valid()¶
Returns:True if it’s valid, False otherwise
Return type:bool

Check if given Midgard.Validable instance is marked as valid. Implementation shall not validate given instance.

New in version 10.05.5.

validate()¶
Raises:GLib.Error

Validate given instance. Implementation shall validate given instance, and in case when it’s invalid, particular error shoudl be set. If instance is valid, error should be None.

New in version 10.05.5.
