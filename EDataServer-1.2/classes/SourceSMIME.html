set_signing_algorithm(signing_algorithm)¶
Parameters:

signing_algorithm (str or None) – the signing algorithm for outgoing messages, or None

Sets the name of the hash algorithm used to digitally sign outgoing messages.

The internal copy of signing_algorithm is automatically stripped of leading and trailing whitespace. If the resulting string is empty, None is set instead.

New in version 3.6.
