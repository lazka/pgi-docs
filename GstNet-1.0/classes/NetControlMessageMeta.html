class GstNet.NetControlMessageMeta¶

GstNet.NetControlMessageMeta can be used to store control messages (ancillary data) which was received with or is to be sent alongside the buffer data. When used with socket sinks and sources which understand this meta it allows sending and receiving ancillary data such as unix credentials (See Gio.UnixCredentialsMessage) and Unix file descriptions (See Gio.UnixFDMessage).
