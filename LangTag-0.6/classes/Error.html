All the fields in the LangTag.Error structure are private to the LangTag.Error implementation.

classmethod new()¶

Creates LangTag.Error object. this function is protected and not supposed to use in applications directly. Use lt_error_set().

clear()¶

is_set(type)¶
Parameters:type (LangTag.ErrorType) – a #LtErrorype_t
Returns:True if any, otherwise False
Return type:int

Checks if self contains type of errors. if #LT_ERR_ANY is set to type, all the types of the errors are targeted. otherwise the result is filtered out by type.

ref()¶

unref()¶

Decreases the reference count of self. when its reference count drops to 0, the object is finalized (i.e. its memory is freed).
