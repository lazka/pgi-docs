class Amtk.MenuShell(**kwargs)¶
Bases:GObject.Object
Abstract:No
Structure:Amtk.MenuShellClass

Amtk.MenuShell extends the Gtk.MenuShell abstract class with the Amtk.MenuShell ::menu-item-selected and Amtk.MenuShell ::menu-item-deselected convenience signals.

One possible use-case is to push/pop longer descriptions of menu items to a Gtk.Statusbar, exactly like Amtk.ApplicationWindow.connect_menu_to_statusbar() does.

Return type:Amtk.MenuShell

Returns the Amtk.MenuShell of gtk_menu_shell. The returned object is guaranteed to be the same for the lifetime of gtk_menu_shell.

New in version 2.0.
