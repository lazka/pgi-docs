WebKit2.WebsiteDataAccessPermissionRequestClass

A permission request for accessing website data from third-party domains.

WebKit2.WebsiteDataAccessPermissionRequest represents a request for permission to allow a third-party domain access its cookies.

When a WebKit2.WebsiteDataAccessPermissionRequest is not handled by the user, it is denied by default.

New in version 2.30.
