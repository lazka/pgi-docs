Bases: GObject.GEnum

Tile loading state.

NONE = 0¶

Initial or undefined state

LOADING = 1¶

Tile is loading

LOADED = 2¶

Tile is loaded but not yet displayed

DONE = 3¶

Tile loading finished. Also used to inform map sources that tile loading has been cancelled.
