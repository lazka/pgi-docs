  • fd (int) – dmabuf file descriptor
  • size (int) – memory size
Returns:

a Gst.Memory based on allocator. When the buffer will be released dmabuf allocator will close the fd. The memory is only mmapped on gst_buffer_mmap() request.

Gst.Memory
