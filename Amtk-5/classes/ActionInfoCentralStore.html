Structure:Amtk.ActionInfoCentralStoreClass

Amtk.ActionInfoCentralStore is a singleton class containing the aggregation of all Amtk.ActionInfoStore’s. Each time an Amtk.ActionInfo is added to an Amtk.ActionInfoStore, it is also added to the Amtk.ActionInfoCentralStore.
