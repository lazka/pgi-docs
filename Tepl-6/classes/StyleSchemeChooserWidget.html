set_style_scheme_id(style_scheme_id)¶
Parameters

style_scheme_id (str) – the new value.

Sets the Tepl.StyleSchemeChooserWidget :tepl-style-scheme-id property.

The GtkSource.StyleScheme is taken from the default GtkSource.StyleSchemeManager as returned by GtkSource.StyleSchemeManager.get_default().

New in version 5.0.

Property Details¶

Tepl.StyleSchemeChooserWidget.props.tepl_style_scheme_id¶
Name

tepl-style-scheme-id

Type

str

Default Value

''

Flags

READABLE, WRITABLE

The GtkSource.StyleSchemeChooser :style-scheme ID, as a string. This property is useful for binding it to a Gio.Settings key.

When the GtkSource.StyleSchemeChooser :style-scheme is None, this property contains the empty string.

New in version 5.0.
