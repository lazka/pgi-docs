  ShapeRendererFunc (cr, attr, do_path, *data)

Details¶

PangoCairo.ShapeRendererFunc(cr, attr, do_path, *data)¶
Parameters:
  • cr (cairo.Context) – a Cairo context with current point set to where the shape should be rendered
  • attr (Pango.AttrShape) – the Pango.AttrType.SHAPE to render
  • do_path (bool) – whether only the shape path should be appended to current path of cr and no filling/stroking done. This will be set to True when called from PangoCairo.layout_path() and PangoCairo.layout_line_path() rendering functions.
  • data (object or None) – user data passed to PangoCairo.context_set_shape_renderer()

Function type for rendering attributes of type Pango.AttrType.SHAPE with Pango’s Cairo renderer.
