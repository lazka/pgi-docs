Signal Details¶

Hinawa.FwUnit.signals.bus_update(fw_unit)¶
Signal Name:bus-update
Flags:RUN_LAST
Parameters:fw_unit (Hinawa.FwUnit) – The object which received the signal

When IEEE 1394 bus is updated, the ::bus-update signal is generated. Handlers can read current generation in the bus via ‘generation’ property.

Signal Name:disconnected
Flags:RUN_LAST
Parameters:fw_unit (Hinawa.FwUnit) – The object which received the signal

When phicical FireWire devices are disconnected from IEEE 1394 bus, the Hinawa.FwUnit becomes unlostening and emits this signal.
