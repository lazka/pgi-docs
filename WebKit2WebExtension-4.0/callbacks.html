Details¶

WebKit2WebExtension.WebExtensionInitializeFunction(extension)¶
Parameters:extension (WebKit2WebExtension.WebExtension) – a WebKit2WebExtension.WebExtension

Type definition for a function that will be called to initialize the web extension when the web process starts.

WebKit2WebExtension.WebExtensionInitializeWithUserDataFunction(extension, user_data)¶
Parameters:
  • extension (WebKit2WebExtension.WebExtension) – a WebKit2WebExtension.WebExtension
  • user_data (GLib.Variant) – a GLib.Variant

Type definition for a function that will be called to initialize the web extensions when the web process starts, and which receives as additional argument the user data set with webkit_web_context_set_web_extensions_initialization_user_data().

New in version 2.4.
