set_current_memory(memory)¶
Parameters

memory (int) – The current amount of RAM in kilobytes.

Sets the current amount of RAM allocated to self in kilobytes (i.e. blocks of 1024 bytes). This can be set to less than the maximum domain memory to allow to balloon the guest memory on the fly. Be aware that libvirt will set it automatically if it’s not explictly set, which means you may need to set this value in addition to ‘memory’ if you want to change the available domain memory after creation.
