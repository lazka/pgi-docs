Gdk.EventPadGroupMode¶

Fields¶

Name Type Access Description
group int r/w the pad group that is switching mode. A Gdk.InputSource.TABLET_PAD device may have one or more groups containing a set of buttons/rings/strips each.
mode int r/w The new mode of group. Different groups in a Gdk.InputSource.TABLET_PAD device may have different current modes.
send_event int r/w True if the event was sent explicitly.
time int r/w the time of the event in milliseconds.
type Gdk.EventType r/w the type of the event (Gdk.EventType.PAD_GROUP_MODE).
window Gdk.Window r/w the window which received the event.
