ref()¶
Returns:The passed WebKit2.ITPThirdParty
Return type:WebKit2.ITPThirdParty

Atomically increments the reference count of self by one. This function is MT-safe and may be called from any thread.

New in version 2.30.

unref()¶

Atomically decrements the reference count of self by one. If the reference count drops to 0, all memory allocated by WebKit2.ITPThirdParty is released. This function is MT-safe and may be called from any thread.

New in version 2.30.
