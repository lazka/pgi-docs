render_icon(widget, size)¶
Parameters

  • widget (Gtk.Widget) – widget of which style settings may be used.

  • size (int) – size of the rendered icon.

Returns

A rendered pixbuf. The pixbuf belongs to self and should not be unreffed.

Return type

GdkPixbuf.Pixbuf

Renders an icon of given size. The base of the icon is set by the last call to one of: GtkSource.MarkAttributes.set_pixbuf(), GtkSource.MarkAttributes.set_gicon() or GtkSource.MarkAttributes.set_icon_name(). size cannot be lower than 1.
