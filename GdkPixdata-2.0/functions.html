Details¶

GdkPixdata.pixbuf_from_pixdata(pixdata, copy_pixels)¶
Parameters:

  • pixdata (GdkPixdata.Pixdata) – a GdkPixdata.Pixdata to convert into a GdkPixbuf.

  • copy_pixels (bool) – whether to copy raw pixel data; run-length encoded pixel data is always copied.

Raises:

GLib.Error

Returns:

a new pixbuf

Return type:

GdkPixbuf.Pixbuf

Converts a GdkPixdata to a GdkPixbuf.

If copy_pixels is TRUE or if the pixel data is run-length-encoded, the pixel data is copied into newly-allocated memory; otherwise it is reused.

Deprecated since version 2.32: Use GResource instead.
