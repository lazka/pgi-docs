WebKit2.NotificationPermissionRequestClass

A permission request for displaying web notifications.

WebKit2.NotificationPermissionRequest represents a request for permission to decide whether WebKit should provide the user with notifications through the Web Notification API.

When a WebKit2.NotificationPermissionRequest is not handled by the user, it is denied by default.

New in version 2.8.
