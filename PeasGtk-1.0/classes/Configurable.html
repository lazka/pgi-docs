Bases:GObject.GInterface

Interface for configurable plugins.

create_configure_widget()¶
Returns:A Gtk.Widget used for configuration.
Return type:Gtk.Widget

Creates the configure widget for the plugin. The returned widget should allow configuring all the relevant aspects of the plugin, and should allow instant-apply, as promoted by the Gnome Human Interface Guidelines.

PeasGtk.PluginManager will embed the returned widget into a dialog box, but you shouldn’t take this behaviour for granted as other implementations of a plugin manager UI might do otherwise.

This method should always return a valid Gtk.Widget instance, never None.
