Mx.Toolbar.signals.close_button_clicked(toolbar)¶
Signal Name:close-button-clicked
Flags:RUN_LAST
Parameters:toolbar (Mx.Toolbar) – The object which received the signal
Returns:True to prevent the parent stage being destroyed.
Return type:bool

Emitted when the close button of the toolbar is clicked.

Normally, the parent stage will be closed when the close button is clicked. Return True from this handler to prevent the stage from being destroyed.
