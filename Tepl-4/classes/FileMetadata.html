load(cancellable)¶
Parameters:cancellable (Gio.Cancellable or None) – optional Gio.Cancellable object, None to ignore.
Raises:GLib.Error
Returns:whether the metadata was loaded successfully.
Return type:bool

Loads synchronously the metadata from Tepl.File :location. The loaded metadata values can then be accessed with Tepl.FileMetadata.get().

If the metadata are loaded successfully, this function deletes all previous metadata stored in the self object memory.

The file at Tepl.File :location, if non-None, must exist on the filesystem, otherwise an error is returned.

If Tepl.File :location is None, False is simply returned.

New in version 1.0.

The asynchronous version of Tepl.FileMetadata.load().

If the metadata is loaded from the metadata manager (i.e. not with Gio.Vfs), this function loads the metadata synchronously. A future version might fix this.

set(key, value)¶
Parameters:
  • key (str) – the name of the metadata.
  • value (str or None) – the value of the metadata, or None to unset.

Sets the value of a metadata. It’s preferable that key starts with a namespace, to not get metadata conflicts between applications.

This function just stores the new metadata value in the self object memory.

New in version 1.0.
