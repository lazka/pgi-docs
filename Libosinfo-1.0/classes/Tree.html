Raises:

Returns:

a new Libosinfo.Tree , or None on error
Return type:

Libosinfo.Tree

Creates a new Libosinfo.Tree for installation tree at location. The location could be any URI that GIO can handle or a local path.

NOTE: Currently this only works for trees with a .treeinfo file

get_treeinfo_arch()¶
Returns:the system id, or None
Return type:str

If self is an ISO9660 image/device, this function retrieves the expected system ID.

Note: In practice, this will usually not be the exact copy of the system ID string on the ISO image/device but rather a regular expression that matches it.
