Details¶

TrackerMiner.MINER_DBUS_INTERFACE = 'org.freedesktop.Tracker1.Miner'¶

The name of the D-Bus interface to use for all data miners that inter-operate with Tracker.

New in version 0.8.

TrackerMiner.MINER_DBUS_NAME_PREFIX = 'org.freedesktop.Tracker1.Miner.'¶

D-Bus name prefix to use for all data miners. This allows custom miners to be written using TrackerMiner.MINER_DBUS_NAME_PREFIX + “Files” for example and would show up on D-Bus under “org.freedesktop.Tracker1.Miner.Files”.

New in version 0.8.

TrackerMiner.MINER_DBUS_PATH_PREFIX = '/org/freedesktop/Tracker1/Miner/'¶

D-Bus path prefix to use for all data miners. This allows custom miners to be written using TrackerMiner.MINER_DBUS_PATH_PREFIX + “Files” for example and would show up on D-Bus under “/org/freedesktop/Tracker1/Miner/Files”.

New in version 0.8.

TrackerMiner.MINER_ERROR_DOMAIN = 'TrackerMiner'¶

Used as the domain for any GLib.Errors reported by TrackerMiner objects.

New in version 0.8.
