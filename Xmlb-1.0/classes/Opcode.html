Parameters:func (int) – a function index
Returns:a Xmlb.Opcode

classmethod text_new(str)¶
Parameters:str (str) – a string
Returns:a Xmlb.Opcode
Return type:Xmlb.Opcode

Creates a new text literal opcode. The str argument is copied internally and is not tied to the lifecycle of the Xmlb.Opcode.

New in version 0.1.1.

classmethod text_new_static(str)¶
Parameters:str (str) – a string
Returns:a Xmlb.Opcode
Return type:Xmlb.Opcode

Creates a new text literal opcode, where str is either static text or will outlive the Xmlb.Opcode lifecycle.

New in version 0.1.1.

classmethod text_new_steal(str)¶
Parameters:str (str) – a string
Returns:a Xmlb.Opcode
Return type:Xmlb.Opcode

Creates a new text literal opcode, stealing the str. Once the opcode is finalized GLib.free() will be called on str.

New in version 0.1.1.
