Details¶

WebKit.URISchemeRequestCallback(request, *user_data)¶
Parameters:

  • request (WebKit.URISchemeRequest) – the WebKit.URISchemeRequest

  • user_data (object or None) – user data passed to the callback

Type definition for a function that will be called back when an URI request is made for a user registered URI scheme.
