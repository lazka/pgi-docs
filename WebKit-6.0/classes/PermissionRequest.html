WebKit.PermissionRequestInterface

A permission request.

There are situations where an embedder would need to ask the user for permission to do certain types of operations, such as switching to fullscreen mode or reporting the user’s location through the standard Geolocation API. In those cases, WebKit will emit a WebKit.WebView ::permission-request signal with a WebKit.PermissionRequest object attached to it.

Allow the action which triggered this request.
