classmethod new(book_cache, sexp, sort_fields, sort_types)¶
Parameters:

  • book_cache (EDataBook.BookCache) – An EDataBook.BookCache

  • sexp (str or None) – search expression; use None or an empty string to get all stored contacts

  • sort_fields ([EBookContacts.ContactField]) – An array of EBookContacts.ContactField(s) as sort keys in order of priority

  • sort_types ([EBookContacts.BookCursorSortType]) – An array of EBookContacts.BookCursorSortTypes, one for each field in sort_fields

Raises:

GLib.Error

Returns:

A newly created EDataBook.BookCacheCursor

Return type:

EDataBook.BookCacheCursor

Creates a new EDataBook.BookCacheCursor.

The cursor should be freed with EDataBook.BookCache.cursor_free() when no longer needed.
