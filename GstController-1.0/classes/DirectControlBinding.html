GstController.DirectControlBindingClass

A value mapping object that attaches control sources to gobject properties. It will map the control values directly to the target property range. If a non-absolute direct control binding is used, the value range [0.0 â¦ 1.0] is mapped to full target property range, and all values outside the range will be clipped. An absolute control binding will not do any value transformations.
