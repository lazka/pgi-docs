classmethod new(name, object, position, sticky)¶
Parameters

  • name (str) – Unique name for identifying the dock object.

  • object (Gdl.DockObject) – Corresponding Gdl.DockObject

  • position (Gdl.DockPlacement) – The position to dock a new item in object

  • sticky (bool) – True if the placeholder move with the object

Returns

The newly created placeholder.

Return type

Gtk.Widget

Creates a new dock placeholder at object place. This is a kind of marker allowing you to dock new items later at this place. It is not completely working though.

Deprecated since version 3.6.
