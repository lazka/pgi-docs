WebKit.WebsiteDataAccessPermissionRequestClass

A permission request for accessing website data from third-party domains.

WebKit.WebsiteDataAccessPermissionRequest represents a request for permission to allow a third-party domain access its cookies.

When a WebKit.WebsiteDataAccessPermissionRequest is not handled by the user, it is denied by default.

New in version 2.30.
