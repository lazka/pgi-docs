GimpUi.DrawableChooserClass

The chooser contains an optional label and a button which queries the core process to pop up a drawable selection dialog.

New in version 3.0.

classmethod new(title, label, drawable_type, drawable)¶
Parameters:

  • title (str or None) – Title of the dialog to use or None to use the default title.

  • label (str or None) – Button label or None for no label.

  • drawable_type (GObject.GType) – the acceptable subtype of choosable drawables.

  • drawable (Gimp.Drawable or None) – Initial drawable.

Returns:

A [class`GimpUi`.DrawableChooser.

Return type:

Gtk.Widget

Creates a new Gtk.Widget that lets a user choose a drawable which must be of type drawable_type. drawable_type of values GObject.TYPE_NONE and %GIMP_TYPE_DRAWABLE are equivalent. Otherwise it must be a subtype of %GIMP_TYPE_DRAWABLE.

When drawable is None, initial choice is from context.

New in version 3.0.
