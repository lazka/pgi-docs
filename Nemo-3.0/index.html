Nemo 3.0 (4.8.6)¶

Parent Project:nemo (Nemo component system facilities)
Description:Nemo is the official file manager for the Cinnamon desktop. It allows one to browse directories, preview files and launch applications associated with them. It is also responsible for handling the icons on the Cinnamon desktop. It works on local and remote filesystems.
Homepage:http://cinnamon.linuxmint.com/
Debian Package:gir1.2-nemo-3.0
Dependencies:
GLib 2.0
Gio 2.0
Gtk 3.0
