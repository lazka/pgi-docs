get_language()¶
Returns:

the selected Gspell.Language, or None if no dictionaries are available.

Return type:

Gspell.Language or None

get_language_code()¶
Returns:

the Gspell.LanguageChooser :language-code. It cannot be None.

Return type:

str

set_language(language)¶
Parameters:

language (Gspell.Language or None) – a Gspell.Language or None to pick the default language.

Sets the selected language.

set_language_code(language_code)¶
Parameters:

language_code (str or None) – a language code, or the empty string or None to pick the default language.

Gspell.LanguageChooser.props.language_code¶
Name:

language-code

Type:

str

Default Value:

''

Flags:

READABLE, WRITABLE

The empty string if the default language was set and the selection hasn’t changed. Or the language code if an explicit language was set or if the selection has changed.
