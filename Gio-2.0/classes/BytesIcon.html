Gio.BytesIcon specifies an image held in memory in a common format (usually png) to be used as icon.

New in version 2.38.

classmethod new(bytes)[source]¶
Parameters

bytes (GLib.Bytes) – a GLib.Bytes.

Returns

a Gio.Icon for the given bytes.

Return type

Gio.BytesIcon

Creates a new icon for a bytes.

This cannot fail, but loading and interpreting the bytes may fail later on (for example, if Gio.LoadableIcon.load() is called) if the image is invalid.

New in version 2.38.
