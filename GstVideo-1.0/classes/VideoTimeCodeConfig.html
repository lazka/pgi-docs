Details¶

class GstVideo.VideoTimeCodeConfig¶

Supported frame rates: 30000/1001, 60000/1001 (both with and without drop frame), and integer frame rates e.g. 25/1, 30/1, 50/1, 60/1.

The configuration of the time code.

New in version 1.10.
