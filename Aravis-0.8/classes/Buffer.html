class Aravis.Buffer(**kwargs)¶
Bases:GObject.Object
Abstract:No
Structure:Aravis.BufferClass

Aravis.Buffer provides a class for the instantiation of buffers used for the storage of the separate images of the video stream. The actual data space may either be allocated by Aravis.Buffer during an object instatiation, of preallocated. Aravis.Buffer also allows the transmission of image metadata, such as offsets and size of the transmitted region of interrest, pixel format and time stamp.

Aravis.Buffer

Creates a new buffer for the storage of the video stream images. The data space can be either preallocated, and the caller is responsible for it’s deallocation, or allocated by this function. If it is the case, data memory will be freed when the buffer is destroyed.

New in version 0.2.0.

Returns:

a new Aravis.Buffer object

Aravis.Buffer

Creates a new buffer for the storage of the video stream images. The data space can be either preallocated, and the caller is responsible for it’s deallocation, or allocated by this function. If it is the case, data memory will be freed when the buffer is destroyed.

If user_data_destroy_func is non None, it will be called in order to destroy user_data when the buffer is destroyed.

New in version 0.2.0.
