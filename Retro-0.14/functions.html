Retro.g_log(sender, log_domain, log_level, message)¶
Parameters:
  • sender (Retro.Core) – a Retro.Core
  • log_domain (str or None) – the log domain, usually GLib.LOG_DOMAIN, or None
  • log_level (GLib.LogLevelFlags) – the log level, either from GLib.LogLevelFlags or a user-defined level
  • message (str) – the message to log

Logs an error or debugging message.

This is a convenience function to forward the log signal of Retro.Core to g_log(), see it for more information.
