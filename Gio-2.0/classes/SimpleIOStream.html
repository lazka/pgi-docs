Gio.SimpleIOStream creates a Gio.IOStream from an arbitrary Gio.InputStream and Gio.OutputStream. This allows any pair of input and output streams to be used with Gio.IOStream methods.

This is useful when you obtained a Gio.InputStream and a Gio.OutputStream by other means, for instance creating them with platform specific methods as Gio.UnixInputStream.new() or g_win32_input_stream_new(), and you want to take advantage of the methods provided by Gio.IOStream.

New in version 2.44.
