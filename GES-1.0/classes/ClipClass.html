create_track_element GES.CreateTrackElementFunc r Method to create the core GES.TrackElement of a clip of this class. If a clip of this class may create several track elements per track type, this should be left as None, and create_track_elements() should be used instead. Otherwise, you should implement this class method and leave create_track_elements() as the default implementation
