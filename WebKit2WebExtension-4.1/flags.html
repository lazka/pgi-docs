class WebKit2WebExtension.HitTestResultContext(value)¶

Bases: GLib.Flags

Enum values with flags representing the context of a WebKit2WebExtension.HitTestResult.

SELECTION = 128¶

a selected element.

New in version 2.8.

MEDIA = 16¶

a video or audio element.

DOCUMENT = 2¶

anywhere in the document.

EDITABLE = 32¶

an editable element

LINK = 4¶

a hyperlink element.

SCROLLBAR = 64¶

a scrollbar element.

IMAGE = 8¶

an image element.
