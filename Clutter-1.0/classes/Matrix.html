class Clutter.Matrix¶

A type representing a 4x4 matrix.

It is identicaly to Cogl.Matrix.

New in version 1.12.

classmethod alloc()¶
Returns:the newly allocated Clutter.Matrix
Return type:Clutter.Matrix

Allocates enough memory to hold a Clutter.Matrix.

New in version 1.12.

free()¶

Frees the memory allocated by Clutter.Matrix.alloc().

New in version 1.12.

init_identity()¶
Returns:the initialized Clutter.Matrix
Return type:Clutter.Matrix

Initializes self with the identity matrix, i.e.:

.xx = 1.0, .xy = 0.0, .xz = 0.0, .xw = 0.0
.yx = 0.0, .yy = 1.0, .yz = 0.0, .yw = 0.0
.zx = 0.0, .zy = 0.0, .zz = 1.0, .zw = 0.0
.wx = 0.0, .wy = 0.0, .wz = 0.0, .ww = 1.0

New in version 1.12.
