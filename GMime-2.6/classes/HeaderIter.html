class GMime.HeaderIter¶

A message or mime-part header iterator. All members should be considered private.

copy_to(dest)¶
Parameters:dest (GMime.HeaderIter) – a GMime.HeaderIter

Copies self to dest.

equal(iter2)¶
Parameters:iter2 (GMime.HeaderIter) – a GMime.HeaderIter
Returns:True if self and iter2 refer to the same header or False otherwise.
Return type:bool

Check that self and iter2 reference the same header.

first()¶
Returns:True on success or False otherwise.
Return type:bool

Updates self to point to the first header.

free()¶

Frees a dynamically-allocated GMime.HeaderIter as created by g_mime_header_iter_new() or g_mime_header_iter_copy().

get_name()¶
Returns:the header name or None if invalid.
Return type:str

Gets the current header’s name.

get_value()¶
Returns:the header’s raw, unprocessed value or None if invalid.

Note: The returned value should be decoded with a function such as GMime.utils_header_decode_text() before displaying to the user.
Return type:str

Gets the current header’s value.

is_valid()¶
Returns:True if self is still valid or False otherwise.
Return type:bool

Checks if a GMime.HeaderIter is valid. An iterator may become invalid if the GMime.HeaderList that the iterator refers to changes.

last()¶
Returns:True on success or False otherwise.
Return type:bool

Updates self to point to the last header.

next()¶
Returns:True on success or False otherwise.
Return type:bool

Advances to the next header.

prev()¶
Returns:True on success or False otherwise.
Return type:bool

Advances to the previous header.

remove()¶
Returns:True on success or False otherwise.
Return type:bool

Removes the current header and advances to the next header.

Note: If you remove the last header in the list, then self will become invalid regardless of whether or not other headers remain.

set_value(value)¶
Parameters:value (str) – new header value
Returns:True if the value was set or False otherwise (indicates invalid iter).

Note: value should be encoded with a function such as GMime.utils_header_encode_text().
Return type:bool

Sets the current header’s value to the new value.
