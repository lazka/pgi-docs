WebKit.GeolocationManagerClass

Geolocation manager.

WebKit.GeolocationManager provides API to get the geographical position of the user. Once a WebKit.GeolocationPermissionRequest is allowed, when WebKit needs to know the user location WebKit.GeolocationManager ::start signal is emitted. If the signal is handled and returns True, the application is responsible for providing the position every time itâs updated by calling WebKit.GeolocationManager.update_position(). The signal WebKit.GeolocationManager ::stop will be emitted when location updates are no longer needed.
