GtkLayerShell 0.1 (0.5.2)¶

Parent Project:gtk-layer-shell (Wayland Layer Shell protocol desktop component library)
Description:A library to write GTK applications that use Layer Shell. This library only makes sense on Wayland compositors that support Layer Shell, and will not work on X11. It supports all Layer Shell features including popups and popovers (GTK popups Just Work™). Please open issues for any bugs you come across.
Homepage:https://github.com/wmww/gtk-layer-shell
Debian Package:gir1.2-gtklayershell-0.1
Dependencies:
Gtk 3.0
