Only meant to be used when the pipewire proxy destroyed signal is triggered.

get_associated_proxy(proxy_type)¶
Parameters

proxy_type (GObject.GType) – a Wp.Proxy subclass GObject.GType

Returns

the associated proxy of the specified proxy_type, or None if there is no association to such a proxy

Return type

Wp.Proxy or None

An associated proxy is a Wp.Proxy subclass instance that is somehow related to this item. For example:

  • An exported Wp.SiEndpoint should have at least:

  • an associated Wp.SiEndpoint

  • an associated WpSession

  • In cases where the item wraps a single PipeWire node, it should also have an associated Wp.Node
