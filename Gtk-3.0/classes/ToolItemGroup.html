Class Details¶

class Gtk.ToolItemGroup(**kwargs)¶
Bases:Gtk.Container, Gtk.ToolShell
Abstract:No
Structure:Gtk.ToolItemGroupClass

A Gtk.ToolItemGroup is used together with Gtk.ToolPalette to add Gtk.ToolItems to a palette like container with different categories and drag and drop support.

CSS nodes

Gtk.ToolItemGroup has a single CSS node named toolitemgroup.

New in version 2.20.
