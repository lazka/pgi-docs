Name of the photo-filename property which can be stored on a Camel.CipherCertInfo.

New in version 3.22.

Camel.CIPHER_CERT_INFO_PROPERTY_SIGNERS_ALT_EMAILS = 'signers-alt-emails'¶

A string containing a list of email addresses of all signers including their alternative emails. Use Camel.Address.unformat() to break them back into separate addresses. This can be set only on the first signer of the first validity, even the addresses can belong to a different signer.

New in version 3.28.
