class Gst.Tracer(**kwargs)¶
Bases:Gst.Object
Abstract:Yes
Structure:Gst.TracerClass

Tracing modules will subclass Gst.Tracer and register through gst_tracer_register(). Modules can attach to various hook-types - see gst_tracing_register_hook(). When invoked they receive hook specific contextual data, which they must not modify.
