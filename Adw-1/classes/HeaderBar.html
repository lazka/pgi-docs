AdwHeaderBarâs CSS node is called headerbar. It contains a windowhandle subnode, which contains a box subnode, which contains two widget subnodes at the start and end of the header bar, each of which contains a box subnode with the .start and .end style classes respectively, as well as a center node that represents the title.

Each of the boxes contains a windowcontrols subnode, see [class`Gtk`.WindowControls] for details, as well as other children.
