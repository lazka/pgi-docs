class Gtk.VScrollbar(*args, **kwargs)¶
Bases:Gtk.Scrollbar
Abstract:No
Structure:Gtk.VScrollbarClass

The Gtk.VScrollbar widget is a widget arranged vertically creating a scrollbar. See Gtk.Scrollbar for details on scrollbars. Gtk.Adjustment pointers may be added to handle the adjustment of the scrollbar or it may be left None in which case one will be created for you. See Gtk.Scrollbar for a description of what the fields in an adjustment represent for a scrollbar.

Gtk.VScrollbar has been deprecated, use Gtk.Scrollbar instead.
