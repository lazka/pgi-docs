GstCheck.CheckLogFilterFunc(log_domain, log_level, message, *user_data)¶
Parameters

  • log_level (GLib.LogLevelFlags) – the log level of the message

  • message (str) – the message that has occurred

  • user_data (object or None) – user data

Returns

True if message should be discarded by GstCheck.
