get_alias()¶
Returns:

An alias

Return type:

GMenu.TreeAlias

This method may only be called if GMenu.TreeIter.next() returned GMenu.TreeItemType.ALIAS.

next()¶
Returns:

The type of the next item that can be retrived from the iterator

Return type:

GMenu.TreeItemType

Change the iterator to the next item, and return its type. If there are no more items, GMenu.TreeItemType.INVALID is returned.
