Retrieve the current user logged using the https://graph.facebook.com/me Graph API function. See gfbgraph_user_get_my_async() for the asynchronous version of this call.

classmethod get_me_async(authorizer, cancellable, callback, *user_data)¶
Parameters:
  • authorizer (GFBGraph.Authorizer) – a GFBGraph.Authorizer.
  • cancellable (Gio.Cancellable or None) – An optional Gio.Cancellable object, or None.
  • callback (Gio.AsyncReadyCallback or None) – A Gio.AsyncReadyCallback to call when the request is completed.
  • user_data (object or None) – The data to pass to callback.

Asynchronously retrieve the current user logged. See GFBGraph.User.get_me() for the synchronous version of this call.

When the operation is finished, callback will be called. You can then call gfbgraph_user_get_me_finish() to get the GFBGraph.User for to the current user logged.
