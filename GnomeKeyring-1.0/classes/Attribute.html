class GnomeKeyring.Attribute¶

An item attribute. Set string if data type is GnomeKeyring.AttributeType.STRING or integer if data type is GnomeKeyring.AttributeType.UINT32

Deprecated since version ???: libsecret only supports string attributes.

Parameters:
  • attributes (GLib.Array) – A #GnomeKeyringAttributeList
  • name (str) – The name of the new attribute
  • value (str) – The value to store in attributes

Store a key-value-pair with a string value in attributes.

Deprecated since version ???: libsecret stores attributes as a GLib.HashTable containing string keys and values, use GLib.HashTable.replace() instead.

get_string()¶
Returns:The value.string pointer of self. This is not a copy, do not free.
Return type:str

Return the string value. It is an error to call this method if self.type is not GnomeKeyring.AttributeType.STRING. This method is mostly useful for language bindings which do not provide union access. In C you should just use attribute->value.string.

Deprecated since version ???: Not needed when using libsecret.

Return the uint32 value. It is an error to call this method if self.type is not GnomeKeyring.AttributeType.UINT32. This method is mostly useful for language bindings which do not provide union access. In C you should just use attribute->value.integer.
