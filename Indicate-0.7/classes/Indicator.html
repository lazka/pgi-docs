Class Details¶

class Indicate.Indicator(**kwargs)¶
Bases:GObject.Object
Abstract:No
Structure:Indicate.IndicatorClass

The indicator object represents a single item that is shared over the indicator bus. This could be something like one IM, one e-mail or a single system update. It should be accessed only through its accessors.

set_displayed(displayed)¶
Parameters:displayed (bool) – Whether or not the indicator is visible to users

Sets whether or not the indicator is visible to the user from a listener. This does not include things like whether the menu is open, but more whether it’s in the menu to be found.

set_property(key, data)¶
Parameters:

Sets a simple string property on self. If the property had previously been set it will replace it with the new value, otherwise it will create the property. This will include an emition of Indicate.Indicator ::modified if the property value was changed.

show()¶

Shows this indicator on the bus. If the Indicate.Server that it’s connected to is not shown itself this function will show the server as well using Indicate.Server.show.

user_display(timestamp)¶
Parameters:timestamp (int) – The time that the event happened

Emits the Indicate.Indicator ::user-display signal simliar to a user clicking on self over the bus. Signal will not be sent if the self is not visible.

Indicate.Indicator.signals.displayed(indicator, arg1)¶
Signal Name:

displayed
Flags:

RUN_LAST
Parameters:
  • indicator (Indicate.Indicator) – The object which received the signal
  • arg1 (bool) – Whether the indicator has been displayed

This is the signal that the indicator has been displayed, or hidden by a listener. In most cases, the signal will be that it has been displayed as most folks don’t go hiding it later.

Indicate.Indicator.signals.hide(indicator)¶
Signal Name:hide
Flags:RUN_LAST
Parameters:indicator (Indicate.Indicator) – The object which received the signal

Emitted every time this indicator is hidden. This is mostly used by Indicate.Server.

Typically this results in an emition of Indicate.Server ::indicator-removed.
