get_task()¶
Returns:the Gio.Task for TrackerMiner.DecoratorInfo on success or None if there is no existing Gio.Task.
Return type:Gio.Task

Get the Gio.Task associated with retrieving extended metadata and information for a URN in Tracker.

The task object’s data (accessible with Gio.Task.get_task_data()) is the #TrackerSparqlBuilder. Use TrackerMiner.DecoratorInfo.complete() to complete the task instead of using this object.

New in version 0.18.
