class GLib.IConv¶

The GLib.IConv struct wraps an iconv() conversion descriptor. It contains private data and should only be accessed using the following functions.

_(inbuf, inbytes_left, outbuf, outbytes_left)[source]¶
Parameters:
  • inbuf (str) – bytes to convert
  • inbytes_left (int) – inout parameter, bytes remaining to convert in inbuf
  • outbuf (str) – converted output bytes
  • outbytes_left (int) – inout parameter, bytes available to fill in outbuf
Returns:

count of non-reversible conversions, or -1 on error
Return type:

int

Same as the standard UNIX routine iconv(), but may be implemented via libiconv on UNIX flavors that lack a native implementation.

GLib provides GLib.convert() and GLib.locale_to_utf8() which are likely more convenient than the raw iconv wrappers.

close()[source]¶
Returns:-1 on error, 0 on success
Return type:int

Same as the standard UNIX routine iconv_close(), but may be implemented via libiconv on UNIX flavors that lack a native implementation. Should be called to clean up the conversion descriptor from g_iconv_open() when you are done converting things.

GLib provides GLib.convert() and GLib.locale_to_utf8() which are likely more convenient than the raw iconv wrappers.
