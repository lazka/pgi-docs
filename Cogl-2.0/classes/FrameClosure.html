class Cogl.FrameClosure¶

An opaque type that tracks a Cogl.FrameCallback and associated user data. A Cogl.FrameClosure pointer will be returned from Cogl.Onscreen.add_frame_callback() and it allows you to remove a callback later using Cogl.Onscreen.remove_frame_callback().

New in version 1.14.
