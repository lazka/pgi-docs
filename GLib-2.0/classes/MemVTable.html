class GLib.MemVTable¶

A set of functions used to perform memory allocation. The same GLib.MemVTable must be used for all allocations in the same program; a call to GLib.mem_set_vtable(), if it exists, should be prior to any use of GLib.

This functions related to this has been deprecated in 2.46, and no longer work.
