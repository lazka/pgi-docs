add_window(window)¶
Parameters:window (Mx.Window) – The Mx.Window to add to the application

Adds a window to the list of windows associated with self. If this is the first window, it will be treated as the primary window and used for startup notification.

This function does not take a reference on window.

is_running()¶
Returns:True if the application is running
Return type:bool

Query whether Mx.Application is running. This will also return True if the given Mx.Application is single instance and there is an instance already running.

quit()¶

Stop the application from running and quit the main loop. This will cause the call to Mx.Application.run() to complete.

remove_action(name)¶
Parameters:name (str) – name of the action to remove

Remove the action with the specified name from the application.

remove_window(window)¶
Parameters:window (Mx.Window) – an Mx.Window

Remove the specified window from the application. This will cause the window to be unreferenced and destroyed unless another reference is held on it.

run()¶

Run the main loop of the application and start processing events. This function will not return until the application is quit. If the application is single instance and an existing instance is already running, this will cause the existing instance to be raised and the function will complete immediately.
