GstGL.GLAllocationParamsCopyFunc(src, dest)¶
Parameters:
  • src (GstGL.GLAllocationParams) – the source GstGL.GLAllocationParams to copy from
  • dest (GstGL.GLAllocationParams) – the source GstGL.GLAllocationParams to copy

Copies the parameters from src into dest. The subclass must compose copy functions from the superclass.

GstGL.GLAllocationParamsFreeFunc(params)¶
Parameters:params (object or None) – a GstGL.GLAllocationParams

Free any dynamically allocated data. The subclass must call the superclass’ free.

GstGL.GLBaseMemoryAllocatorAllocFunction(allocator, params)¶
Parameters:
  • allocator (GstGL.GLBaseMemoryAllocator) – a GstGL.GLBaseMemoryAllocator
  • params (GstGL.GLAllocationParams) – the GstGL.GLAllocationParams to allocate the memory with
Returns:

a newly allocated GstGL.GLBaseMemory from allocator and params
Return type:

GstGL.GLBaseMemory
