class ECal.ReminderData¶

Contains data related to single reminder occurrence.

New in version 3.30.

classmethod free(rd)¶
Parameters:rd (object or None) – an ECal.ReminderData, or None

Frees previously allocated ECal.ReminderData structure with ECal.ReminderData.new() or ECal.ReminderData.copy(). The function does nothing when rd is None.

New in version 3.30.
