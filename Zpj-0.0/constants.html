Zpj.SKYDRIVE_FOLDER_MY_DOCUMENTS = 'me/skydrive/my_documents'¶

Friendly name for the Documents folder. It can be used as the Zpj.SkydriveEntry :id and #ZpjSkydriveEntry:parent_id.
