map_id(info, buffer, id, flags)[source]¶
Parameters:
  • info (GstVideo.VideoInfo) – a GstVideo.VideoInfo
  • buffer (Gst.Buffer) – the buffer to map
  • id (int) – the frame id to map
  • flags (Gst.MapFlags) – Gst.MapFlags
Returns:

True on success.
Return type:

bool

Use info and buffer to fill in the values of self with the video frame information of frame id.

When id is -1, the default frame is mapped. When id != -1, this function will return False when there is no GstVideo.VideoMeta with that id.

All video planes of buffer will be mapped and the pointers will be set in self->data.
