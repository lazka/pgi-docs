Gtk.HSV is the âcolor wheelâ part of a complete color selector widget. It allows to select a color by determining its HSV components in an intuitive way. Moving the selection around the outer ring changes the hue, and moving the selection point inside the inner triangle changes value and saturation.

Gtk.HSV has been deprecated together with Gtk.ColorSelection, where it was used.
