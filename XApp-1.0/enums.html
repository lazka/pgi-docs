class XApp.StatusIconState(value)¶

Bases: GObject.GEnum

NATIVE = 0¶

The XApp.StatusIcon is currently being handled by an XApp.StatusIconMonitor (usually in an applet).

FALLBACK = 1¶

The XApp.StatusIcon is currently being handled by a legacy system tray implementation (using Gtk.StatusIcon).

NO_SUPPORT = 2¶

The XApp.StatusIcon is not currently being handled by any kind of status icon implementation.
