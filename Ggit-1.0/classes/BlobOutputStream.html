get_id()¶
Raises

GLib.Error

Returns

a Ggit.OId or None.

Return type

Ggit.OId or None

Get the id of the written blob. The blob id is only available after the stream has been properly closed. If an error occurred while writing the blob, the None is returned and error is set accordingly.
