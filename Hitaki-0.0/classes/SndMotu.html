Hitaki.SndMotuClass

A GObject.Object-derived object for sound unit of MOTU FireWire series. The [class`SndMotu`] is an object class derived from [class`GObject`.Object] with protocol implementation for Mark of the Unicorn (MOTU) FireWire series supported by ALSA firewire-motu driver (snd-firewire-motu).
