Hitaki.SndTascamClass

A GObject.Object-derived object for sound unit of TASCAM FireWire series. The [class`SndTascam`] is an object class derived from [class`GObject`.Object] with protocol implementation for TASCAM FireWire series supported by ALSA firewire-tascam driver (snd-firewire-tascam). The image of state consists of 64 quadlets according to SNDRV_FIREWIRE_TASCAM_STATE_COUNT in UAPI of ALSA firewire stack.
