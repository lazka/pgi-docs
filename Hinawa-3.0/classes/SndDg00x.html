Hinawa.SndDg00xClass

A notification listener for Digidesign Digi 00x models. A [class`SndDg00x`] listen to Dg00x notification and generates signal when received.

Deprecated since version 2.5.: libhitaki library provides [class`Hitaki`.SndDigi00x] as the alternative.
