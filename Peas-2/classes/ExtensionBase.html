Structure:

Peas.ExtensionBaseClass

Base class for C extensions.

Peas.ExtensionBase can optionally be used as a base class for the extensions of your plugin. By inheriting from it, you will make your extension able to access the related [class`PluginInfo`], and especially the location where all the data of your plugin lives.

Non-C extensions will usually not inherit from this class: Python plugins automatically get a “plugin_info” attribute that serves the same purpose.
