Infinoted 0.7 (0.7.2)¶

Parent Project:libinfinity (infinote-based collaborative editing (Gtk widgets))
Description:libinfinity is library to build collaborative text editors. Changes to the text buffers are synced to all other clients over a central server. Even though a central server is involved, the local user sees his changes applied instantly and the merging is done on the individual clients.
