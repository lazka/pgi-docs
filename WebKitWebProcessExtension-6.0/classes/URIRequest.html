WebKitWebProcessExtension.URIRequestClass

Represents a URI request.

A WebKitWebProcessExtension.URIRequest can be created with a URI using the WebKitWebProcessExtension.URIRequest.new() method, and you can get the URI of an existing request with the WebKitWebProcessExtension.URIRequest.get_uri() one.
