Gdk.DmabufTextureClass

A GdkTexture representing a DMA buffer.

To create a GdkDmabufTexture, use the auxiliary [class`Gdk`.DmabufTextureBuilder] object.

Dma-buf textures can only be created on Linux.

New in version 4.14.
