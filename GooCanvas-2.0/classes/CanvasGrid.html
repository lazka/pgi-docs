Class Details¶

class GooCanvas.CanvasGrid(**kwargs)¶
Bases:GooCanvas.CanvasItemSimple
Abstract:No
Structure:GooCanvas.CanvasGridClass

GooCanvas.CanvasGrid represents a grid item. A grid consists of a number of equally-spaced horizontal and vertical grid lines, plus an optional border.

It is a subclass of GooCanvas.CanvasItemSimple and so inherits all of the style properties such as “stroke-color”, “fill-color” and “line-width”.

It also implements the GooCanvas.CanvasItem interface, so you can use the GooCanvas.CanvasItem functions such as GooCanvas.CanvasItem.raise_() and GooCanvas.CanvasItem.rotate().

To create a GooCanvas.CanvasGrid use goo_canvas_grid_new().

To get or set the properties of an existing GooCanvas.CanvasGrid, use g_object_get() and g_object_set().

The grid’s position and size is specified with the GooCanvas.CanvasGrid :x, GooCanvas.CanvasGrid :y, GooCanvas.CanvasGrid :width and GooCanvas.CanvasGrid :height properties.

The GooCanvas.CanvasGrid :x-step and GooCanvas.CanvasGrid :y-step properties specify the distance between grid lines. The GooCanvas.CanvasGrid :x-offset and GooCanvas.CanvasGrid :y-offset properties specify the distance before the first grid lines.

The horizontal or vertical grid lines can be hidden using the GooCanvas.CanvasGrid :show-horz-grid-lines and GooCanvas.CanvasGrid :show-vert-grid-lines properties.

The width of the border can be set using the GooCanvas.CanvasGrid :border-width property. The border is drawn outside the area specified with the GooCanvas.CanvasGrid :x, GooCanvas.CanvasGrid :y, GooCanvas.CanvasGrid :width and GooCanvas.CanvasGrid :height properties.

Other properties allow the colors and widths of the grid lines to be set. The grid line color and width properties override the standard GooCanvas.CanvasItemSimple :stroke-color and GooCanvas.CanvasItemSimple :line-width properties, enabling different styles for horizontal and vertical grid lines.
