Gsk.Renderer

Creates a new Broadway renderer.

The Broadway renderer is the default renderer for the broadway backend. It will only work with broadway surfaces, otherwise it will fail the call to Gsk.Renderer.realize().

This function is only available when GTK was compiled with Broadway support.
