CredentialsPrompterLoopPromptFunc (prompter, source, credentials, user_data, cancellable)

Details¶

EDataServerUI.CredentialsPrompterLoopPromptFunc(prompter, source, credentials, user_data, cancellable)¶
Parameters

  • prompter (EDataServerUI.CredentialsPrompter) – an EDataServerUI.CredentialsPrompter

  • source (EDataServer.Source) – an EDataServer.Source, as passed to EDataServerUI.CredentialsPrompter.loop_prompt_sync()

  • credentials (EDataServer.NamedParameters) – an EDataServer.NamedParameters with provided credentials

  • user_data (object or None) – user data, as passed to EDataServerUI.CredentialsPrompter.loop_prompt_sync()

  • cancellable (Gio.Cancellable or None) – a Gio.Cancellable, as passed to EDataServerUI.CredentialsPrompter.loop_prompt_sync()

Returns

True to continue the loop (reprompt credentials), unless authenticated is also set to True, or False on error, as an indication that the loop should be terminated.

out_authenticated

set to True, when the authentication was successful

Return type

(bool, out_authenticated: bool)
