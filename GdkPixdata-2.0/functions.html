Details¶

GdkPixdata.pixbuf_from_pixdata(pixdata, copy_pixels)[source]¶
Parameters:
  • pixdata (GdkPixdata.Pixdata) – a GdkPixdata.Pixdata to convert into a GdkPixbuf.Pixbuf.
  • copy_pixels (bool) – whether to copy raw pixel data; run-length encoded pixel data is always copied.
Raises:

GLib.Error
Returns:

a new GdkPixbuf.Pixbuf.
Return type:

GdkPixbuf.Pixbuf

Converts a GdkPixdata.Pixdata to a GdkPixbuf.Pixbuf. If copy_pixels is True or if the pixel data is run-length-encoded, the pixel data is copied into newly-allocated memory; otherwise it is reused.

Deprecated since version 2.32: Use Gio.Resource instead.
