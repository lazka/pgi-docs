The Gtk.HScale widget is used to allow the user to select a value using a horizontal slider. To create one, use Gtk.HScale.new_with_range().

The position to show the current value, and the number of decimal places shown can be set using the parent Gtk.Scale classâs functions.

Gtk.HScale has been deprecated, use Gtk.Scale instead.

Creates a new horizontal scale widget that lets the user input a number between min and max (including min and max) with the increment step. step must be nonzero; itâs the distance the slider moves when using the arrow keys to adjust the scale value.
