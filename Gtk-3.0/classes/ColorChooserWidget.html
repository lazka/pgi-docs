Gtk.ColorChooserWidgetClass

The Gtk.ColorChooserWidget widget lets the user select a color. By default, the chooser presents a predefined palette of colors, plus a small number of settable custom colors. It is also possible to select a different color with the single-color editor. To enter the single-color editing mode, use the context menu of any color of the palette, or use the â+â button to add a new custom color.

The chooser automatically remembers the last selection, as well as custom colors.

To change the initially selected color, use Gtk.ColorChooser.set_rgba(). To get the selected color use Gtk.ColorChooser.get_rgba().

The Gtk.ColorChooserWidget is used in the Gtk.ColorChooserDialog to provide a dialog for selecting colors.

CSS names

Gtk.ColorChooserWidget has a single CSS node with name colorchooser.

New in version 3.4.
