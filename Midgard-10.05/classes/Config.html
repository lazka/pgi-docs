static list_files(user)¶
Parameters:user (bool) – boolean switch for system or user’s config files
Returns:newly allocated and None terminated array of file names.
Return type:[str]

List all available configuration files. If user value is set to True, all available files from ~/.midgard/conf.d will be listed. Only system files ( usually from /etc/midgard/conf.d ) will be listed if user value is set to False.

Returned array should be freed when no longer needed.

read_file(filename, user)¶
Parameters:
  • filename (str) – name of the file to read
  • user (bool) – boolean switch for system or user’s config files
Raises:

GLib.Error
Returns:

True when file has been read , False otherwise.
Return type:

bool

This method reads configuration file from the given name and sets Midgard.Config object’s properties. Such initialized Midgard.Config instance may be reused among midgard-core and midgard-php extension for example, without any need to re-read configuration file and without any need to re-initalize Midgard.Config object instance.

Set True as user boolean value to read files from user’s home directory.

New in version 9.3.

Return type:

bool

Saves configuration file for the given Midgard.Config.

This method saves configuration file with the given name. If third user parameter is set to True, then configuration file will be saved in ~/.midgard2/conf.d directory.

User’s conf.d directory will be created if doesn’t exist.
