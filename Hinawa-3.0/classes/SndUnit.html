Hinawa.SndUnitClass

An event listener for ALSA FireWire sound devices. This class is an application of ALSA FireWire stack. Any functionality which ALSA drivers in the stack can be available.

Deprecated since version 2.5.: libhitaki library provides [class`Hitaki`.SndUnit] as the alternative.

Signal Details¶

Hinawa.SndUnit.signals.disconnected(snd_unit)¶
Signal Name

disconnected

Flags

RUN_LAST, DEPRECATED

Parameters

snd_unit (Hinawa.SndUnit) – The object which received the signal

Emitted when the sound card is not available anymore due to unbinding driver or hot unplugging. The owner of this object should call [method`GObject`.Object.unref] as quickly as possible to release ALSA hwdep character device.

New in version 2.0.

Deprecated since version 2.5.: Use change notify signal of [property`Hitaki`.AlsaFirewire:is-disconnected] instead.
