GSSDP.ResourceBrowser

target is a generic string the resource browser listens for on the SSDP bus. There are several possible targets such as

  • “ssdp:all” for everything
  • “upnp:rootdevice” for UPnP device entry points, not caring about the device type
  • The UUID of a specific device
  • Device types such as “urn:schemas-upnp-org:py:data::device<GSSDP.ResourceBrowser.props.device>:MediaServer:1”
  • Service types such as “urn:schemas-upnp-org:py:data::service<GSSDP.ResourceBrowser.props.service>:ContentDirectory:1”
