do_create_source() virtual¶
Returns

The source element to use.

Return type

Gst.Element

Creates the Gst.Element to put in the source topbin. Other elements will be queued, like a volume. In the case of a AudioUriSource, for example, the subclass will return a decodebin, and we will append a volume.

New in version 1.20.
