class GtkLayerShell.KeyboardMode(value)¶

Bases: GLib.Enum

GtkLayerShell.KeyboardMode.NONE: This window should not receive keyboard events. GtkLayerShell.KeyboardMode.EXCLUSIVE: This window should have exclusive focus if it is on the top or overlay layer. GtkLayerShell.KeyboardMode.ON_DEMAND: The user should be able to focus and unfocues this window in an implementation defined way. Not supported for protocol version < 4. GtkLayerShell.KeyboardMode.ENTRY_NUMBER: Should not be used except to get the number of entries. (NOTE: may change in future releases as more entries are added)
