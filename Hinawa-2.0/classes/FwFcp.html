transaction(req_frame, resp_frame)¶
Parameters:
  • req_frame (bytes) – An array with elements for request byte data. The value of this argument should point to the array and immutable.
  • resp_frame (bytes) – An array with elements for response byte data. Callers should give it for buffer with enough space against the request since this library performs no reallocation. Due to the reason, the value of this argument should point to the pointer to the array and immutable. The content of array is mutable.
