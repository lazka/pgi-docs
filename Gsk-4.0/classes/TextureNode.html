classmethod new(texture, bounds)¶
Parameters

  • bounds (Graphene.Rect) – the rectangle to render the texture into

Returns

A new GskRenderNode

Return type

Gsk.TextureNode

Creates a GskRenderNode that will render the given texture into the area given by bounds.

Note that GSK applies linear filtering when textures are scaled and transformed. See [class`Gsk`.TextureScaleNode] for a way to influence filtering.
