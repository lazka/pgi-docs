WebKit2.PointerLockPermissionRequestClass

A permission request for locking the pointer.

WebKit2.PointerLockPermissionRequest represents a request for permission to decide whether WebKit can lock the pointer device when requested by web content.

When a WebKit2.PointerLockPermissionRequest is not handled by the user, it is allowed by default.

New in version 2.28.
