Wp.new_files_iterator(dirs, subdir, suffix)Â¶
Parameters

  • dirs (Wp.LookupDirs) â the directories to look into

  • subdir (str or None) â the name of the subdirectory to search in, inside the configuration directories

  • suffix (str or None) â The filename suffix, None matches all entries

Returns

a new iterator iterating over strings which are absolute paths to the configuration files found

Return type

Wp.Iterator

Creates an iterator to iterate over configuration files in the subdir of the configuration directories.

Files are sorted across the hierarchy of configuration and data directories with files in higher-priority directories shadowing files in lower-priority directories. Files are only checked for existence, a caller must be able to handle read errors. the iterator may contain directories too; it is the responsibility of the caller to ignore or recurse into those.

New in version 0.4.2.
