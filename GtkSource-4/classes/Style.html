GtkSource.StyleClass

apply(tag)¶
Parameters

tag (Gtk.TextTag) – a Gtk.TextTag to apply styles to.

This function modifies the Gtk.TextTag properties that are related to the GtkSource.Style properties. Other Gtk.TextTag properties are left untouched.

If self is non-None, applies self to tag.

If self is None, the related *-set properties of Gtk.TextTag are set to False.

New in version 3.22.
