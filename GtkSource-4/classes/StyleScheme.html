get_style(style_id)¶
Parameters:style_id (str) – id of the style to retrieve.
Returns:style which corresponds to style_id in the self, or None when no style with this name found. It is owned by self and may not be unref’ed.
Return type:GtkSource.Style or None

New in version 2.0.
