get_data_out()¶
Returns:bytes
Return type:int

Gets the mutable data of the chunk.

WARNING: At the moment FwupdPlugin.Chunk.get_data_out() returns the same data as FwupdPlugin.Chunk.get_data() in all cases. The caller should verify the data passed to FwupdPlugin.Chunk.array_new() is also writable (i.e. not const or mmap) before using this function.
