  • name (str or None) – A bus name (well-known or unique) or None if connection is not a message bus connection.
  • object_path (str) – An object path.
  • cancellable (Gio.Cancellable or None) – A Gio.Cancellable or None.
  • callback (Gio.AsyncReadyCallback or None) – A Gio.AsyncReadyCallback to call when the request is satisfied.
  • user_data (object or None) – User data to pass to callback.

Asynchronously creates a proxy for the D-Bus interface ‘org.gnome.SessionManager.Presence [gdbus-interface-org-gnome-SessionManager-Presence.top_of_page]’. See Gio.DBusProxy.new() for more details.

When the operation is finished, callback will be invoked in the thread-default main loop of the thread you are calling this method from. You can then call CScreensaver.SessionPresenceProxy.new_finish() to get the result of the operation.

See CScreensaver.SessionPresenceProxy.new_sync() for the synchronous, blocking version of this constructor.
