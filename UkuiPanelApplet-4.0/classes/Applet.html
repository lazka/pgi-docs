set_background_widget(widget)¶
Parameters:widget (Gtk.Widget) – a Gtk.Widget.

Configure #PanelApplet to automatically draw the background of the applet on widget. It is generally enough to call this function with self as widget.

Deprecated since version 3.20: Do not use this API. Since 3.20 this function does nothing.
