classmethod new()¶
Returns:A GnomeBluetooth.FilterWidget widget

Note: Must call GnomeBluetooth.FilterWidget.bind_filter () to bind the GnomeBluetooth.FilterWidget to a GnomeBluetooth.Chooser.
Return type:Gtk.Widget

Creates a new GnomeBluetooth.FilterWidget which can be bound to a GnomeBluetooth.Chooser to control filtering of that GnomeBluetooth.Chooser. Usually used in conjunction with a GnomeBluetooth.Chooser which has the “has-internal-filter” property set to False.

bind_filter(chooser)¶
Parameters:chooser (GnomeBluetooth.Chooser) – The GnomeBluetooth.Chooser widget to bind the filter to.

Binds a GnomeBluetooth.FilterWidget to a GnomeBluetooth.Chooser such that changing the GnomeBluetooth.FilterWidget results in filters being applied on the GnomeBluetooth.Chooser. Any properties set on a bound GnomeBluetooth.Chooser will also be set on the GnomeBluetooth.FilterWidget.

set_title(title)¶
Parameters:title (str) – Title for the GnomeBluetooth.FilterWidget.

Used to set a different title for the GnomeBluetooth.FilterWidget than the default.
