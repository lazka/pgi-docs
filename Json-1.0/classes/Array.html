class Json.Array¶

foreach_element(func, *data)¶
Parameters:
  • func (Json.ArrayForeach) – the function to be called on each element
  • data (object or None) – data to be passed to the function

Iterates over all elements of self and calls func on each one of them.

It is safe to change the value of a Json.Node of the self from within the iterator func, but it is not safe to add or remove elements from the self.

New in version 0.8.

hash()¶
Returns:hash value for self
Return type:int

Calculate a hash value for the given self (a Json.Array).

The hash is calculated over the array and all its elements, recursively. If the array is immutable, this is a fast operation; otherwise, it scales proportionally with the length of the array.

New in version 1.2.

remove_element(index_)¶
Parameters:index (int) – the position of the element to be removed

Removes the Json.Node inside self at index_ freeing its allocated resources.

seal()¶

Seals the Json.Array, making it immutable to further changes. This will recursively seal all elements in the array too.

If the self is already immutable, this is a no-op.

New in version 1.2.

unref()¶

Decreases by one the reference count of a Json.Array. If the reference count reaches zero, the array is destroyed and all its allocated resources are freed.
