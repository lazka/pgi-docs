write()¶
Raises:GLib.Error

Return type:GLib.Bytes

Writes the image, which will try to call a superclassed ->write() function.

By default (and in most cases) this just provides the value set by the FwupdPlugin.FirmwareImage.set_bytes() function.

New in version 1.3.3.

write_chunk(address, chunk_sz_max)¶
Parameters:
  • address (int) – an address greater than dfu_element_get_address()
  • chunk_sz_max (int) – the size of the new chunk
Raises:

GLib.Error
Returns:

a GLib.Bytes, or None
Return type:

GLib.Bytes

Gets a block of data from the image. If the contents of the image is smaller than the requested chunk size then the GLib.Bytes will be smaller than chunk_sz_max. Use FwupdPlugin.common_bytes_pad() if padding is required.

If the address is larger than the size of the image then an error is returned.

New in version 1.3.1.
