class GdkPixdata.Pixdata¶

A GdkPixdata.Pixdata contains pixbuf information in a form suitable for serialization and streaming.

deserialize(stream)[source]¶
Parameters:stream (bytes) – stream of bytes containing a serialized GdkPixdata.Pixdata structure.
Raises:GLib.Error
Returns:Upon successful deserialization True is returned, False otherwise.
Return type:bool

Deserializes (reconstruct) a GdkPixdata.Pixdata structure from a byte stream. The byte stream consists of a straightforward writeout of the GdkPixdata.Pixdata fields in network byte order, plus the pixel_data bytes the structure points to. The self contents are reconstructed byte by byte and are checked for validity. This function may fail with GdkPixbuf.PixbufError.CORRUPT_IMAGE or GdkPixbuf.PixbufError.UNKNOWN_TYPE.

Deprecated since version 2.32: Use Gio.Resource instead.

serialize()[source]¶
Returns:A newly-allocated string containing the serialized GdkPixdata.Pixdata structure.

Serializes a GdkPixdata.Pixdata structure into a byte stream. The byte stream consists of a straightforward writeout of the GdkPixdata.Pixdata fields in network byte order, plus the pixel_data bytes the structure points to.

Deprecated since version 2.32: Use Gio.Resource instead.
