class AppStream.CacheFlags(value)¶

Bases: GLib.Flags

Flags on how caching should be used.

NONE = 0¶

No flags.

USE_USER = 1¶

Create an user-specific metadata cache.

USE_SYSTEM = 2¶

Use and - if possible - update the global metadata cache.

NO_CLEAR = 4¶

Don’t clear the cache when opening it.
