Handy.ComboRowGetEnumValueNameFunc(value, *user_data)¶
Parameters:
  • value (Handy.EnumValueObject) – the value from the enum from which to get a name
  • user_data (object or None) – user data
Returns:

a newly allocated displayable name that represents value
Return type:

str

Called for combo rows that are bound to an enumeration with Handy.ComboRow.set_for_enum() for each value from that enumeration.

Handy.ComboRowGetNameFunc(item, *user_data)¶
Parameters:
  • item (GObject.Object) – the item from the model from which to get a name
  • user_data (object or None) – user data
Returns:

a newly allocated displayable name that represents item
Return type:

str

Called for combo rows that are bound to a Gio.ListModel with Handy.ComboRow.bind_name_model() for each item that gets added to the model.
