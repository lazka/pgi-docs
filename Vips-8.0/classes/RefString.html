class Vips.RefString¶
classmethod new(str)¶
Parameters:str (str) – string to store
Returns:the new Vips.RefString, or None on error.
Return type:Vips.RefString

Create a new refstring. These are reference-counted immutable strings, used to store string data in vips image metadata.

Strings must be valid utf-8; use blob for binary data.

See also: Vips.Area.unref().

get(length)¶
Parameters:length (int or None) – return length here, optionally
Returns:The C string held by self.
Return type:str

Get a pointer to the private string inside a refstr. Handy for language bindings.

See also: Vips.value_get_ref_string().
