activate()¶
Returns

True if sucessfully activated, False otherwise.

Return type

bool

Activates the plugin by calling activate() virtual method. All plugins should derive their classes from this virtual class and implement this method. If the plugin implements IAnjutaPreferences, it is prompted to install it’s preferences.

add_watch(name, added, removed, *user_data)¶
Parameters

  • name (str) – Name of the value to watch.

  • added (Anjuta.PluginValueAdded) – Callback to call when the value is added.

  • removed (Anjuta.PluginValueRemoved) – Callback to call when the value is removed.

  • user_data (object or None) – User data to pass to callbacks.

Returns

Watch ID.

Return type

int

Adds a watch for name value. When the value is added in shell, the added callback will be called and when it is removed, the removed callback will be called. The returned ID is used to remove the watch later.

Parameters
