set_accumulation_enabled(enable)¶
Parameters:enable (bool) – True to enable an accumulation buffer

Sets whether the accumulation buffer is enabled. When enabled, an extra offscreen buffer is allocated, and the contents of the offscreen texture are blended with this accumulation buffer. By default, the blend function is set to blend the contents of the offscreen texture with the accumulation buffer at the opacity specified in the alpha component of the blend constant. This opacity is 50% by default.

New in version 1.2.

set_auto_update(auto_update)¶
Parameters:auto_update (bool) – True if the offscreen surface should be automatically updated

Enable automatic updating of the offscreen surface when the child is updated.

set_child(actor)¶
Parameters:actor (Clutter.Actor) – A Clutter.Actor

Redirects the painting of actor to the offscreen surface owned by self. In the event that actor is unparented, it will be parented to self. Note that when you redirect the painting of actor, it will no longer be painted in its original position in the scenegraph.

set_pick_child(pick)¶
Parameters:pick (bool) – True to enable picking of the child actor

Enable picking of the child actor.

set_redirect_enabled(enabled)¶
Parameters:enabled (bool) – True if redirection to the offscreen surface should be enabled

Sets the value of the Mx.Offscreen :redirect-enabled property. When redirection is enabled, the painting of the child of self will be redirected to the texture of self.

New in version 1.2.

update()¶

Updates the offscreen surface. This causes the child of self to be drawn into the texture of self.
