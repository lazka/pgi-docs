get_introspection_async(callback, *user_data)¶
Parameters:
  • callback (GUPnP.ServiceIntrospectionCallback) – callback to be called when introspection object is ready.
  • user_data (object or None) – user_data to be passed to the callback.

Note that introspection object is created from the information in service description document (SCPD) provided by the service so it can not be created if the service does not provide an SCPD.

get_introspection_async_full(callback, cancellable, *user_data)¶
Parameters:
  • callback (GUPnP.ServiceIntrospectionCallback) – callback to be called when introspection object is ready.
  • cancellable (Gio.Cancellable or None) – Gio.Cancellable that can be used to cancel the call, or None.
  • user_data (object or None) – user_data to be passed to the callback.

Note that introspection object is created from the information in service description document (SCPD) provided by the service so it can not be created if the service does not provide an SCPD.

If cancellable is used to cancel the call, callback will be called with error code Gio.IOErrorEnum.CANCELLED.

New in version 0.20.9.

introspect_async(cancellable, callback, *user_data)¶
Parameters:
  • cancellable (Gio.Cancellable or None) – Gio.Cancellable that can be used to cancel the call, or None.
  • callback (Gio.AsyncReadyCallback or None) – callback to be called when introspeciton object is ready.
  • user_data (object or None) – user_data to be passed to the callback.

Note that introspection object is created from the information in service description document (SCPD) provided by the service so it can not be created if the service does not provide an SCPD.

If cancellable is used to cancel the call, callback will be called with error code Gio.IOErrorEnum.CANCELLED.

New in version 1.2.2.
