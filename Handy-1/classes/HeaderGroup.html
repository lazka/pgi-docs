Handy.HeaderGroup.props.decorate_all¶
Name:decorate-all
Type:bool
Default Value:False
Flags:READABLE, WRITABLE, EXPLICIT_NOTIFY

Whether the elements of the group should all receive the full decoration. This is useful in conjunction with Handy.Leaflet :folded when the leaflet contains the header bars of the group, as you want them all to display the complete decoration when the leaflet is folded.

New in version 1.0.
