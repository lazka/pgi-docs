Name:

show-deleted

Type:

bool

Default Value:

False

Flags:

READABLE, WRITABLE

Whether to include deleted contacts in the query feed. Deleted contacts return True from GData.ContactsContact.is_deleted(), and have no other information. They do not normally appear in query results.
