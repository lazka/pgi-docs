classmethod load(path, object)¶
Parameters

  • path (str) – A path of the saved dictionary file.

  • object (GObject.Object or None) – If the GObject.Object has “unicode-deserialize-progress” signal, this function will emit (the number of desrialized IBus.UnicodeData, * the total number of IBus.UnicodeData) of uint values with that signal by 100 times. Otherwise None.

Returns

An IBus.UnicodeData list loaded from the saved cache file.
