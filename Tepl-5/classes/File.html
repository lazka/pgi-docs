get_short_name()¶
Returns:the self short name. Free with GLib.free() when no longer needed.
Return type:str

Gets the self short name. If the Tepl.File :location isn’t None, returns its display-name (see Gio.FILE_ATTRIBUTE_STANDARD_DISPLAY_NAME). Otherwise returns “Untitled File N”, with N the Nth untitled file of the application, starting at 1. When an untitled file is closed or its location is set, its untitled number is released and can be used by a later file.

New in version 5.0.
