get_image(width, height)¶
Parameters

  • width (int) – target width

  • height (int) – target height

Returns

an AppStream.Image, or None

Return type

AppStream.Image

Gets the AppStream.Image closest to the target size. The AppStream.Image may not actually be the requested size, and the application may have to pad / rescale the image to make it fit. Only images for the current active locale (or fallback, if images are not localized) are considered.

New in version 0.14.0.

set_active_locale(locale)¶
Parameters

locale (str or None) – a POSIX or BCP47 locale, or None. e.g. “de_DE”

Set the current active locale, which is used to get localized messages. If the AppStream.Component linking this AppStream.Screenshot was fetched from a localized database, usually only one locale is available.
