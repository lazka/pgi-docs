get_axis_bounds(axis, bounds)¶
Parameters

  • axis (GOffice.GraphAxisType) – GOffice.GraphAxisType

  • bounds (GOffice.GraphPlotBoundInfo) – GOffice.GraphPlotBoundInfo

Returns

The data providing the bound (does not add a ref)

Return type

GOffice.Data

Queries self for its axis preferences for axis and stores the results in bounds. All elements of bounds are initialized to sane values before the query _EXCEPT_ ::fmt. The caller is responsible for initializing it. This is done so that once on plot has selected a format the others need not do the lookup too if so desired.

Caller is responsible for unrefing ::fmt.
